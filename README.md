#### install

```
npm install nps-i
```

or install it globally

```
npm install -g nps-i
```

&nbsp;

#### usage

Replace `nps` with `nps-i` in your package.json

```json
{
  "scripts": {
    "start": "nps-i"
  }
}
```

You can also use it with the shorthand `ni` when installed globally.
